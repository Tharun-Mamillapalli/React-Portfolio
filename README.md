# React-portfolio

A React based personal portfolio app using create-react-app and React Router v4.

# Tools

create-react-app cli. 
React MDL material design  
React Router v4

# Start App

1.Clone repo, install,  
2.cd into folder and run:  
3.npm install.  
4.npm start
